XmlCommentElement Class

This class implements IXmlSerializable and is instantiated in XmlAnnotate. Properties or fields of this type can be implemented in serialized classes when inheriting XmlAnnotate is not possible or undesired. In the constructor its possible to alter the default settings that would otherwise be specified globally with static properties.

Remarks

If serialization occurs against an instance of this class that has been constructed using the default zero parameter constructor, a NotSupportedException will occur. It is necessary to identify the Type to used for annotation prior to serialization.
